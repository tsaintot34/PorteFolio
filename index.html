TERRA GUSTA

Une plateforme de recettes de cuisine du monde.

Contexte & Problématique

Création d'un site web dynamique pour centraliser des recettes de cuisine variées, axé sur une expérience utilisateur (UX) simple et une recherche performante par origine géographique, categories et difficulté. Le défi était de gérer une grande quantité de données tout en garantissant une navigation fluide.

Stack Technique

  • HTML5 & CSS3
  • Design Responsive
  • JavaScript (Gestion DOM & événements)
  • Simulation d'une base de données de recettes (JSON)
  • **Git** (Contrôle de version)

Fonctionnalités Clés

  • **Recherche & Filtres** : Recherche instantanée et filtrage par Titre/pays/Catégories/difficulté.
  • **Affichage dynamique** : Chargement des fiches recettes via JavaScript sans rechargement de page.
  • **Modale de Recette** : Aperçu détaillé d'une recette dans une modale pour améliorer l'UX.
  • **Design Responsive** : Adaptation optimale pour mobiles, tablettes et desktops.

Défis & Apprentissages

Le principal défi a été l'optimisation des performances pour le filtrage et la recherche sur un grand jeu de données. Cela a nécessité de maîtriser les structures de données en JavaScript et d'optimiser les algorithmes de recherche. J'ai aussi perfectionné les techniques de design responsive complexes pour les cartes de recettes.
